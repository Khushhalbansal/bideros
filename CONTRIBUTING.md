# Contributing to BidArena 🏏

First off, thank you for considering contributing to BidArena! It's people like you that make open source such a great community to learn, inspire, and create.

## Where to Start?

We are currently looking to **overhaul and level up our UI/UX** to make the auction experience more cinematic and engaging. If you have experience with animations, 3D effects, or modern frontend design, we'd love your help!

Check out our issues page for open tasks, especially the ones labeled `enhancement`, `ui`, or `help wanted`.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/bidarena.git
   cd bidarena
   ```
3. **Create a new branch** for your feature or bugfix:
   ```bash
   git checkout -b feature/amazing-new-ui
   ```
4. **Make your changes** and commit them with descriptive messages.
5. **Push your branch** to your fork:
   ```bash
   git push origin feature/amazing-new-ui
   ```
6. **Open a Pull Request** against the `main` branch of the original BidArena repository.

## UI/UX Goals

We are specifically aiming to add:
- **Parallax Effects**: Depth to the landing and dashboard pages.
- **3D Elements**: To make the player cards and auction items pop.
- **Water / Particle Effects**: Engaging background aesthetics.
- **Clicking & Page Animations**: Smooth page transitions using Framer Motion.
- **Animated Backgrounds**: To replace static elements and add IPL-level energy.

**Reference Inspiration:**  
Check out [Drake Design](https://www.drakedesign.work/) for the level of interactive, high-end animations and cinematic feel we are aiming to achieve!
